# Funnel Analysis | Python EDA & Power BI Dashboard

## 1. Executive Summary
This project analyzes a customer conversion funnel to understand user behavior across different stages of the customer journey.

Python was used for exploratory data analysis (EDA) to examine patterns and data distributions. The processed data was then used to build an interactive Power BI dashboard for funnel visualization and conversion analysis.

The project identifies conversion rates, stage drop-offs, and potential bottlenecks in the funnel.

---

## 2. Business Problem
Businesses need to understand how users move through the sales or product funnel.

Key questions addressed in this analysis:

- How many users enter each funnel stage?
- Where do users drop off in the conversion process?
- Which stages show the highest loss of potential customers?
- How can the funnel be optimized to improve conversions?

The goal is to identify bottlenecks in the funnel and improve conversion performance.

---

## 3. Methodology

### Data Preparation
- Imported dataset into Python
- Performed exploratory data analysis
- Checked missing values and data consistency
- Analyzed distributions and stage counts
- Created aggregated metrics for funnel stages

### Python Exploratory Data Analysis
EDA was performed using:

- Pandas for data manipulation
- Matplotlib and Seaborn for visualization
- Conversion stage analysis
- Funnel stage counts and drop-off analysis

### Power BI Dashboard
The processed dataset was imported into Power BI.

Dashboard components include:

- Funnel visualization for conversion stages
- Stage-wise user counts
- Conversion rate analysis
- Drop-off identification across stages

Power BI funnel charts help visualize sequential processes and identify performance issues in each stage.

## 5. Results

The funnel dashboard highlights user behavior across multiple conversion stages.

Key findings include:

- Significant drop-offs occur at specific funnel stages.
- Conversion rates vary across stages of the user journey.
- Early funnel stages attract large user volumes, while later stages show reduced conversions.
- Funnel visualization helps identify bottlenecks affecting overall conversion.
